📚 Tentang Aplikasi

ShopeeSentiment

Aplikasi analisis sentimen ulasan produk Shopee berbasis Python untuk mata kuliah Perancangan Aplikasi Sains Data.

📌 Deskripsi Aplikasi

ShopeeSentiment adalah aplikasi web berbasis Python yang memungkinkan pengguna menganalisis sentimen ulasan produk Shopee secara otomatis. Pengguna hanya perlu memasukkan URL produk Shopee, dan sistem akan menganalisis ratusan ulasan untuk memberikan ringkasan lengkap beserta rekomendasi apakah produk tersebut layak dibeli atau tidak.

Aplikasi dirancang menggunakan pendekatan hybrid paradigma — menggabungkan Object-Oriented Programming (OOP) untuk struktur aplikasi dan Functional Programming (FP) untuk transformasi data, sesuai materi Week 5–6 mata kuliah Perancangan Aplikasi Sains Data, Telkom University.

Python 3.10+ Flask Pandas NumPy Matplotlib OOP + FP

🏗️ Object-Oriented Programming

Aplikasi mengimplementasikan tiga prinsip OOP utama sesuai materi perkuliahan:

🔒 Encapsulation

Setiap kelas menyembunyikan atribut private (_filepath, _reviews) dan hanya mengekspos metode publik sebagai antarmuka. Contoh: DataLoader._filepath hanya bisa diakses via load().

🧬 Inheritance

BaseAnalyzer sebagai parent class. RatingAnalyzer dan LexiconAnalyzer mewarisi sebagai child. HybridAnalyzer mewarisi dari LexiconAnalyzer (inheritance bertingkat).

🔄 Polymorphism

Metode analyze() dipanggil identik pada semua analyzer, tetapi perilakunya berbeda-beda: rating-only, lexicon+rating, atau hybrid dengan bobot likes.

⚡ Functional Programming

Tiga prinsip FP diimplementasikan pada lapisan transformasi data:

🧊 Immutability

Kamus kata (STOPWORDS_ID, POSITIVE_WORDS) menggunakan frozenset. Semua pure function selalu mengembalikan objek baru tanpa mengubah argumen input.

🎯 Referential Transparency

score_to_sentiment(5) selalu mengembalikan "positive". Output hanya bergantung pada input, tidak ada state global yang berubah.

🔧 Higher-Order Functions

make_classifier() mengembalikan fungsi. preprocess_pipeline() menggunakan reduce. filter_by_sentiment() menerima fungsi sebagai argumen (map, lambda).

🔄 Pipeline Analisis

Alur kerja dari URL produk hingga laporan hasil:

1
Input URL Produk
Pengguna memasukkan URL produk Shopee melalui form web.
2
Pengambilan Ulasan (Scraping)
ShopeScraper.scrape() mengambil ulasan dari dataset. Dalam produksi: gunakan Selenium/Playwright.
3
Preprocessing Teks (FP Pipeline)
clean_text → tokenize → remove_stopwords via functools.reduce (function composition).
4
Klasifikasi Sentimen (OOP Polymorphism)
Rating sebagai anchor + kamus kata Bahasa Indonesia untuk label positif/netral/negatif.
5
Agregasi & Visualisasi
Statistik rating, distribusi sentimen, keyword extraction, dan pembuatan chart (Matplotlib).
6
Rekomendasi & Laporan
Skor gabungan (60% sentimen + 40% rating) menghasilkan label: Layak Dibeli / Pertimbangkan / Hati-Hati.
